Physics Formulas

Escape Velocity
$$v_e = \sqrt{\frac{2GM}{R}}$$

Energy of Simple Harmonic Oscillators
$$E = K + U$$
$$\text{Pendulum: } E = \frac{1}{2}mv^2 + mgy$$
$$\text{Spring (H): } E = \frac{1}{2}mv^2 + \frac{1}{2}kx^2$$
$$\text{Spring (V): } E = \frac{1}{2}mv^2 + \frac{1}{2}kx^2 + mgy$$

Time and Distance to Stop
$$t = \frac{v}{\mu g}$$
$$d = \frac{v^2}{2\mu g}$$
$$t = \frac{2d}{v_0}$$

Collisions
$$0 = m_1v_{1f} + m_2v_{2f} \text{ (explosion)}$$
$$m_1v_{1i} + m_2v_{2i} = (m_1+m_2)v_f \text{ (inelastic)}$$

Moment of Inertia
$$\text{Point Mass: } I = mr^2$$
$$\text{Solid Disk: } I = \frac{1}{2}mr^2$$
$$\text{Hollow Ring: } I = mr^2$$
$$\text{Rod (center): } I = \frac{1}{12}ml^2$$
$$\text{Rod (end): } I = \frac{1}{3}ml^2$$

Net Torque
$$\tau_{net} = I\alpha$$

Centripetal Acceleration
$$a_c = r\omega^2 = \frac{v^2}{r}$$

Angular Velocity From Frequency
$$\omega = 2\pi f$$

Rolling Object
$$KE_{total} = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$$
$$mgh = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$$

Angular Momentum
$$I_1\omega_1 = I_2\omega_2$$
$$\tau\Delta t = \Delta L = I\Delta\omega$$

Orbital Motion
$$v_{orbit} = \sqrt{\frac{GM}{r}}$$
$$T_{orbit} = \frac{2\pi r^{3/2}}{\sqrt{GM}}$$

Parallel Axis Theorem
$$I_{end} = I_{cm} + md^2$$
$$I_{cm} = I_{end} - md^2$$

Kinematics and Graphs
$$\text{F-t area} = \Delta p$$
$$\text{v-t area} = \Delta x$$
$$\text{F-x area} = \Delta KE$$
$$\text{slope of } \Delta x \text{ vs } t^2 = \frac{a}{2}$$
$$v_{peak} = v_0\cos\theta \text{ (never zero)}$$

Elevator Forces
$$F_N = m(g+a) \text{ (up)}$$
$$F_N = m(g-a) \text{ (down)}$$

Contact Force in Systems
$$F_{contact} = m_{behind} \cdot a$$
$$T = \frac{m_{target} \cdot F_{driver}}{m_{total}}$$

Circular Motion
$$mg - F_N = \frac{mv^2}{r} \text{ (hilltop)}$$
$$F_N - mg = \frac{mv^2}{r} \text{ (dip)}$$
$$F_N + mg = \frac{mv^2}{r} \text{ (loop top)}$$
$$\mu = \frac{v^2}{rg} \text{ (flat turn)}$$
$$T_{conical} = 2\pi\sqrt{\frac{r}{g\tan\theta}} \text{ (mass independent)}$$

Energy
$$\Delta KE = W_{net}$$
$$\Delta KE + \Delta U_g = -W_{friction}$$
$$W_g = mgh \text{ (vertical only, path independent)}$$
$$KE_{max} = \frac{1}{2}kA^2 \text{ (SHM, mass cancels)}$$
$$|\Delta U_2| = |\Delta U_1| + \Delta KE \text{ (Atwood)}$$
$$P = Fv = \frac{mgh}{\Delta t}$$

Momentum
$$J = F\Delta t = \Delta p$$
$$v_{cm} = \frac{p_{total}}{m_{total}} = \text{constant (no ext. force)}$$
$$\frac{\Delta p_{system}}{\Delta t} = F_{net,\ external}$$

Fluids
$$P_1 + \rho gh_1 + \frac{1}{2}\rho v_1^2 = P_2 + \rho gh_2 + \frac{1}{2}\rho v_2^2$$
$$v = \sqrt{2g(D-d)} \text{ (open container)}$$
$$x = \sqrt{4d(D-d)} \text{ (horizontal distance)}$$
$$x \propto \sqrt{h} \text{ (concave down graph)}$$

SHM
$$\omega = \sqrt{\frac{k}{m}}$$
$$T_{spring} = 2\pi\sqrt{\frac{m}{k}}$$
$$T_{pendulum} = 2\pi\sqrt{\frac{L}{g}} \text{ (mass independent)}$$
$$KE_{max} = \frac{1}{2}kA^2$$

Gravity
$$F_g = \frac{Gm_1m_2}{r^2}$$
$$g_{planet} = \frac{GM}{r^2}$$
$$v_e = \sqrt{\frac{2GM}{R}}$$
$$\text{treat } g \text{ as constant only when } h \ll r_{planet}$$

Angular Velocity
$$\omega = 2\pi f$$
$$a_c = r\omega^2 = \frac{v^2}{r}$$

Rotation
$$L = mRv \text{ (point mass on string)}$$
$$\frac{\alpha_B}{\alpha_A} = \frac{I_A}{I_B} \text{ (same torque)}$$
$$a = R\alpha \quad v = R\omega$$
