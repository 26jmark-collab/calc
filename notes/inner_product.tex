Physics Formulas

Escape Velocity
$$v_e = \sqrt{\frac{2GM}{R}}$$

Energy of Simple Harmonic Oscillators
$$E = K + U$$
$$\text{Pendulum: } E = \frac{1}{2}mv^2 + mgy$$
$$\text{Spring (H): } E = \frac{1}{2}mv^2 + \frac{1}{2}kx^2$$
$$\text{Spring (V): } E = \frac{1}{2}mv^2 + \frac{1}{2}kx^2 + mgy$$

Time to Stop From Friction
$$t = \frac{v}{\mu g}$$

Distance to Stop From Friction
$$d = \frac{v^2}{2\mu g}$$

Explosion from Rest
$$0 = m_1v_{1f} + m_2v_{2f}$$

Inelastic Collision
$$m_1v_{1i} + m_2v_{2i} = (m_1 + m_2)v_f$$

Moment of Inertia
$$\text{Point Mass: } I = mr^2$$
$$\text{Solid Disk: } I = \frac{1}{2}mr^2$$
$$\text{Hollow Ring: } I = mr^2$$
$$\text{Rod (center): } I = \frac{1}{12}ml^2$$
$$\text{Rod (end): } I = \frac{1}{3}ml^2$$

Net Torque
$$\tau_{net} = I\alpha$$

Centripetal Acceleration
$$a_c = r\omega^2 = \frac{v^2}{r}$$

Angular Velocity From Frequency
$$\omega = 2\pi f$$

Total KE of Rolling Object
$$KE_{total} = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$$

Conservation of Energy with Rotation
$$mgh = \frac{1}{2}mv^2 + \frac{1}{2}I\omega^2$$

Conservation of Angular Momentum
$$I_1\omega_1 = I_2\omega_2$$

Orbital Speed
$$v = \sqrt{\frac{GM}{r}}$$

Period of Orbit
$$T = \frac{2\pi r^{3/2}}{\sqrt{GM}}$$

Angular Impulse
$$\tau\Delta t = \Delta L = I\Delta\omega$$

Parallel Axis Theorem
$$I_{end} = I_{cm} + md^2$$
$$I_{cm} = I_{end} - md^2$$
