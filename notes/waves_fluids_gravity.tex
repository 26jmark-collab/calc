SHM and Waves

SHM Energy
$$E = K + U$$
$$\text{Pendulum: } E = \frac{1}{2}mv^2 + mgy$$
$$\text{Spring (H): } E = \frac{1}{2}mv^2 + \frac{1}{2}kx^2$$
$$\text{Spring (V): } E = \frac{1}{2}mv^2 + \frac{1}{2}kx^2 + mgy$$
$$KE_{max} = \frac{1}{2}kA^2 \text{ (mass cancels)}$$

Periods
$$\omega = \sqrt{\frac{k}{m}}$$
$$T_{spring} = 2\pi\sqrt{\frac{m}{k}}$$
$$T_{pendulum} = 2\pi\sqrt{\frac{L}{g}}$$
$$T_{conical} = 2\pi\sqrt{\frac{r}{g\tan\theta}}$$

Fluids
$$P_1 + \rho gh_1 + \frac{1}{2}\rho v_1^2 = P_2 + \rho gh_2 + \frac{1}{2}\rho v_2^2$$
$$v = \sqrt{2g(D-d)} \text{ (open container)}$$
$$x = \sqrt{4d(D-d)} \text{ (horizontal distance)}$$
$$x \propto \sqrt{h} \text{ (concave down graph)}$$
$$\text{Set } P_1=P_2 \text{ and } v_1=0 \text{ for open container}$$

Gravity
$$F_g = \frac{Gm_1m_2}{r^2}$$
$$g_{planet} = \frac{GM}{r^2}$$
$$v_e = \sqrt{\frac{2GM}{R}}$$
$$v_{orbit} = \sqrt{\frac{GM}{r}}$$
$$T_{orbit} = \frac{2\pi r^{3/2}}{\sqrt{GM}}$$
$$\text{g constant near surface only}$$
$$\text{use } F_g = \frac{Gm_1m_2}{r^2} \text{ when h is large}$$
